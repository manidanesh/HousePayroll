# Household Payroll System

A personal payroll application for household employers to manage caregiver payroll while maintaining compliance with U.S. federal and Colorado employment requirements.

## Project Overview

This system enables a single household employer to:
- Manage multiple W-2 caregiver employee profiles
- Track daily hours worked per caregiver
- Calculate payroll with accurate tax withholdings
- Generate compliant paystubs in PDF format
- Maintain immutable, auditable payroll records
- Export data for W-2 and Schedule H tax form preparation

## Key Features

- **Multi-caregiver Support**: Manage multiple caregivers with independent payroll processing
- **Tax Compliance**: Automated federal (Social Security, Medicare, FUTA) and Colorado (SUTA) tax calculations
- **Data Integrity**: Immutable finalized records with audit trails and version tracking
- **Legal Compliance**: Colorado-compliant paystub generation
- **Pure Functional Design**: Deterministic payroll calculations with no side effects
- **Export Capabilities**: PDF paystubs and CSV data for tax form preparation
